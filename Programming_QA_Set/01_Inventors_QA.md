# ❓ Q&A — Inventors & History of Programming Languages

---

## Q1. Who invented the C programming language and when?
**Answer:** Dennis Ritchie invented C in **1972** at **Bell Labs** (AT&T).
He also co-created the Unix operating system with Ken Thompson.

---

## Q2. Who invented C++ and what does the ++ mean?
**Answer:** **Bjarne Stroustrup** invented C++ in **1980** at Bell Labs.
The `++` is the increment operator in C — meaning C++ is "one step above C."
He added Object-Oriented Programming features to C.

---

## Q3. Who invented Python and why the name "Python"?
**Answer:** **Guido van Rossum** (Netherlands) invented Python in **1991**.
He named it after the comedy show **"Monty Python's Flying Circus"** — not the snake!

---

## Q4. Who invented Java and what is its famous motto?
**Answer:** **James Gosling** invented Java in **1995** at **Sun Microsystems**.
Motto: **"Write Once, Run Anywhere"** — Java code runs on any device with a JVM.

---

## Q5. Who invented JavaScript and in how many days was it created?
**Answer:** **Brendan Eich** created JavaScript in **1995** at **Netscape**.
It was created in just **10 days**! Originally called "Mocha," then "LiveScript," then JavaScript.

---

## Q6. Who invented HTML and when?
**Answer:** **Tim Berners-Lee** invented HTML in **1991** at **CERN** (Switzerland).
He also invented the World Wide Web (WWW).

---

## Q7. Which was the FIRST high-level programming language?
**Answer:** **FORTRAN** (Formula Translation) — invented by **John W. Backus** at IBM in **1957**.
Used for mathematics and scientific computing.

---

## Q8. Who is called the "Mother of Computing" or "First Programmer"?
**Answer:** **Ada Lovelace** (1815–1852) — she wrote the first algorithm intended for a machine (Babbage's Analytical Engine).
Also, **Grace Hopper** is called "Mother of COBOL" and "First debugger" — she literally found a bug (a moth) in a computer!

---

## Q9. Who invented COBOL and why is it important today?
**Answer:** **Grace Hopper** (with a team) invented COBOL in **1959**.
Stands for **Common Business-Oriented Language**. Still used in **banks and finance systems** today (ATMs, government systems).

---

## Q10. Who invented LISP and why is it famous?
**Answer:** **John McCarthy** at MIT invented LISP in **1958**.
It is the second oldest high-level language still in use. McCarthy also **coined the term "Artificial Intelligence" (AI)**.

---

## Q11. Who created CSS and when?
**Answer:** **Håkon Wium Lie** proposed CSS in **1994** (published 1996 by W3C).
CSS stands for **Cascading Style Sheets** — used to style web pages.

---

## Q12. Who invented SQL and what does it stand for?
**Answer:** **Donald D. Chamberlin** and **Raymond F. Boyce** at IBM invented SQL in **1974**.
SQL = **Structured Query Language** — used to manage databases.

---

## Q13. Who invented Go (Golang) and why?
**Answer:** **Robert Griesemer, Rob Pike, and Ken Thompson** at **Google** created Go in **2009**.
Purpose: Fast, simple, and efficient for large-scale server programs.

---

## Q14. Who invented Rust and who now maintains it?
**Answer:** **Graydon Hoare** created Rust in **2010**, sponsored by **Mozilla**.
Now maintained by the Rust Foundation. Famous for memory safety without garbage collection.

---

## Q15. Who created TypeScript and how is it related to JavaScript?
**Answer:** **Anders Hejlsberg** at **Microsoft** created TypeScript in **2012**.
TypeScript is a **superset of JavaScript** — adds static typing to JS.
Fun fact: Anders also created **C#** and **Turbo Pascal**!

---

## Q16. Name the inventor of Ruby and their philosophy?
**Answer:** **Yukihiro "Matz" Matsumoto** (Japan) created Ruby in **1995**.
Philosophy: **"Principle of Least Surprise"** — Ruby should behave in a way that minimizes confusion.

---

## Q17. Who created PHP and what does it stand for?
**Answer:** **Rasmus Lerdorf** (Denmark/Canada) created PHP in **1994**.
PHP originally stood for "Personal Home Page Tools" — now officially means **PHP: Hypertext Preprocessor** (recursive acronym).

---

## Q18. What are the 5 generations of programming languages?

**Answer:**
| Generation | Type | Example |
|-----------|------|---------|
| 1GL | Machine Language | Binary (0s and 1s) |
| 2GL | Assembly Language | MOV, ADD instructions |
| 3GL | High-Level Language | C, Java, Python |
| 4GL | Very High-Level / Domain-specific | SQL, MATLAB |
| 5GL | AI / Logic-based | Prolog, Mercury |

---

## Q19. Difference between Compiler and Interpreter?

**Answer:**
| Feature | Compiler | Interpreter |
|---------|---------|-------------|
| Translation | Entire program at once | Line by line |
| Speed | Faster execution | Slower execution |
| Error detection | After full compilation | Stops at first error |
| Example | C, C++, Java | Python, JavaScript |
| Output | Object/Machine code | Direct execution |

---

## Q20. Who invented Pascal and what was its purpose?
**Answer:** **Niklaus Wirth** (ETH Zürich, Switzerland) created Pascal in **1970**.
It was designed specifically to **teach students structured programming**.
